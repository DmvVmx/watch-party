# 🚀 Быстрый старт Watch Party

> **Для разработчиков в Сочи и Волгограде** 🏖️❄️

## ⚡ За 5 минут до просмотра фильмов

### Шаг 1: Скачайте проект
```bash
git clone <your-repo-url>
cd watch-party-app
```

### Шаг 2: Установите зависимости
```bash
npm run install-all
```

### Шаг 3: Запустите в разработке
```bash
npm run dev
```

**Готово!** Откройте http://localhost:3000

---

## 🌐 Развертывание на сервере (продакшен)

### VPS/Dedicated Server:

```bash
# Один скрипт установит всё
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```

### Docker (простейший способ):

```bash
# Клонируем
git clone <your-repo-url>
cd watch-party-app

# Запускаем всё одной командой
docker-compose up -d

# Готово! Приложение на порту 3000
```

## 📱 Как пользоваться

1. **Откройте** приложение в браузере
2. **Введите** ваше имя 
3. **Создайте комнату** (кнопка "Создать комнату")
4. **Скопируйте ссылку** и отправьте девушке в Волгоград
5. **Выберите видео** (загрузите файл или вставьте ссылку)
6. **Наслаждайтесь** совместным просмотром! 🍿

### Поддерживаемые форматы видео:
- ✅ MP4 файлы (рекомендуется)
- ✅ Прямые ссылки на видео (.mp4, .webm, .ogg)
- ✅ Локальные файлы до 2GB

## 🔧 Если что-то не работает

### Проблема: "Не могу подключиться"
**Решение:** Проверьте что порт 3000 открыт в файрволе

### Проблема: "Видео не загружается" 
**Решение:** Убедитесь что ссылка ведет напрямую к файлу .mp4

### Проблема: "Нет синхронизации"
**Решение:** Обновите страницы, проверьте интернет соединение

### Проблема: "Чат не работает"
**Решение:** Проверьте WebSocket соединение (должна быть зеленая точка "Подключено")

## 🔗 Полезные команды

```bash
# Разработка
npm run dev              # Запуск в режиме разработки
npm run build           # Сборка для продакшена
npm start              # Запуск в продакшене

# PM2 (на сервере)
pm2 status             # Статус приложений
pm2 logs watch-party   # Просмотр логов
pm2 restart watch-party # Перезапуск
```

## 📞 Поддержка

Если возникли проблемы:

1. **Проверьте логи**: `pm2 logs watch-party`
2. **Перезапустите**: `pm2 restart watch-party`  
3. **Проверьте порт**: `sudo netstat -tlnp | grep 3000`
