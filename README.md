# Watch Party - Приложение для совместного просмотра видео

Современное веб-приложение для совместного просмотра видео с друзьями в режиме реального времени. Аналог Rave с собственным видеоплеером, чатом и синхронизацией.

## ✨ Основные возможности

### 🎬 Видеоплеер
- **HTML5 видеоплеер** с кастомными элементами управления
- **Полная синхронизация** play/pause/seek между всеми участниками
- **Автоподстройка времени** при отставании более чем на 1 сек
- **Кастомные контролы**: play/pause, timeline, громкость, полный экран
- **Красивые состояния**: загрузка, буферизация, ожидание видео
- **Поддержка форматов**: MP4, WebM, OGG

### 🎯 Источники видео
- **Локальные файлы**: загрузка MP4 с вашего устройства
- **Прямые ссылки**: поддержка URL на видеофайлы
- **Проверка доступности** видео перед загрузкой

### 👥 Комнаты и участники
- **Создание комнат** с уникальными ID
- **Приглашения по ссылке** - просто скопируйте и отправьте
- **До 10 участников** в одной комнате
- **Система ролей**: хост управляет видео, участники смотрят
- **Автоназначение** нового хоста при выходе текущего

### 💬 Чат в реальном времени
- **Мгновенные сообщения** через Socket.IO
- **Красивый дизайн** с анимациями
- **История сообщений** до 100 последних
- **Отображение времени** отправки

### 🎨 Современный дизайн
- **Темная тема** с градиентами
- **Glassmorphism эффекты** и размытие
- **Плавные анимации** и переходы
- **Адаптивная верстка** для телефонов и ПК
- **Аккуратные тени** и скругленные углы

## 🚀 Быстрый старт

### Требования
- Node.js 18+ 
- NPM или Yarn

### Установка и запуск

1. **Клонируйте репозиторий**:
```bash
git clone <your-repo-url>
cd watch-party-app
```

2. **Установите зависимости**:
```bash
npm run install-all
```

3. **Запуск в режиме разработки**:
```bash
npm run dev
```
Приложение будет доступно по адресу: http://localhost:3000

4. **Сборка для продакшена**:
```bash
npm run build
```

5. **Запуск в продакшене**:
```bash
npm start
```

## 🌐 Развертывание на сервере

### Подготовка сервера

1. **Установите Node.js 18+** на ваш сервер
2. **Установите PM2** для управления процессами:
```bash
npm install -g pm2
```

### Развертывание

1. **Загрузите код на сервер**:
```bash
git clone <your-repo-url>
cd watch-party-app
```

2. **Установите зависимости**:
```bash
npm run install-all
```

3. **Соберите проект**:
```bash
npm run build
```

4. **Создайте файл ecosystem.config.js** для PM2:
```javascript
module.exports = {
  apps: [{
    name: 'watch-party',
    script: './server/index.js',
    env: {
      NODE_ENV: 'production',
      PORT: 3000
    },
    instances: 1,
    exec_mode: 'cluster'
  }]
}
```

5. **Запустите с PM2**:
```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

### Nginx конфигурация

Создайте файл `/etc/nginx/sites-available/watch-party`:

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        proxy_read_timeout 86400;
    }
}
```

Активируйте конфигурацию:
```bash
sudo ln -s /etc/nginx/sites-available/watch-party /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### SSL сертификат с Let's Encrypt

```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```

## 🛠 Технологии

### Frontend
- **React 18** - современная библиотека для UI

## 🎮 Как использовать

### Создание комнаты
1. Откройте приложение
2. Введите ваше имя
3. Нажмите "Создать комнату"
4. Скопируйте ссылку и отправьте друзьям

### Присоединение к комнате  
1. Получите ссылку от друга
2. Перейдите по ссылке или введите ID комнаты
3. Введите ваше имя
4. Нажмите "Присоединиться"

### Просмотр видео
- **Хост** выбирает видео (файл или ссылку)
- **Хост** управляет воспроизведением
- **Все участники** смотрят синхронно
- **Используйте чат** для общения

## 🔧 API Events (Socket.IO)

### Клиент → Сервер
- `create-room` - создание комнаты
- `join-room` - присоединение к комнате  
- `video-state-change` - изменение состояния видео
- `change-video` - смена видео
- `send-message` - отправка сообщения

### Сервер → Клиент
- `room-updated` - обновление данных комнаты
- `video-sync` - синхронизация видео
- `video-changed` - смена видео
- `new-message` - новое сообщение
- `participants-updated` - обновление участников
